## 一、范围说明

- 一期（MVP）范围：数据接入、病种画像、主力病种分析、漏洞规则、预警闭环、报表导出。
- 二期（增强）范围：标准制定中心、路径变异管理、来源转化评估、绩效联动。
- 本清单默认覆盖一期必须交付 + 二期关键准备项。

## 五、发布门禁（Go/No-Go）

- P0任务全部完成且通过测试。
- P1任务完成率 >= 90%，剩余问题有明确补丁计划。
- UAT关键用例通过率 100%，总体通过率 >= 95%。
- 无阻塞级/严重级缺陷（Sev1/Sev2）。
- 生产回滚预案已演练通过。
